# こども動画アルバム

YouTubeにアップロードした子供の動画を、年別・月別・日別に整理して見返すためのWebアプリです。

動画本体はアプリに保存せず、YouTubeの動画を埋め込み再生します。表示タイトル、撮影日修正、タグ、お気に入り、メモなどのメタデータはブラウザ内に保存します。

## 起動方法

このフォルダでローカルサーバーを起動します。

```powershell
python -m http.server 4173
```

ブラウザで開きます。

```text
http://localhost:4173
```

## 主な機能

- YouTube動画の手動同期
- YouTube URL / 動画IDでの手動追加
- 年月日別の動画一覧
- サムネイル一覧
- 動画詳細での埋め込み再生
- 表示タイトル、撮影日、メモ、タグ、お気に入りの編集
- 撮影日からの年齢表示
- タイトルによる自動カテゴリ分け。1本の動画が複数カテゴリに入ることがあります
- カテゴリ絞り込み
- タイトル、メモ、タグ、年月日検索
- JSONエクスポート / インポート

## YouTube同期の準備

YouTube同期を使うには、Google CloudでOAuthクライアントIDを作成します。

1. Google Cloud Consoleでプロジェクトを作成
2. YouTube Data API v3を有効化
3. OAuth同意画面を設定
4. OAuthクライアントIDを作成
5. アプリケーションの種類は「ウェブアプリケーション」
6. 承認済みのJavaScript生成元に `http://localhost:4173` を追加
7. 作成したクライアントIDをアプリの設定画面に保存

同期時には、管理者のGoogleアカウントでYouTube読み取り権限を許可します。

## 運用方針

- YouTube動画は限定公開を基本にする
- アプリ独自のログイン機能は初期版では使わない
- メタデータは端末ごとのブラウザ内保存
- 子供の生年月は初期値として `2024-11` を設定
- 端末変更やバックアップ用にJSONエクスポートを使う

## 注意点

- ブラウザデータを削除すると、タグやメモなどのメタデータも消えます。
- 夫婦間や端末間でのメタデータ共有は、初期版では未対応です。
- YouTubeから撮影日が取得できない動画は、投稿日を初期値として使います。
- 非公開動画は、視聴や共有の扱いが複雑になるため、初期版では限定公開運用を推奨します。
