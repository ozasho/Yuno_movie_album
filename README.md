# こども動画アルバム

YouTubeにアップロードした子供の動画を、年別・月別・日別に整理して見返すためのWebアプリです。

動画本体はアプリに保存せず、YouTubeの動画を埋め込み再生します。表示タイトル、撮影日修正、タグ、お気に入り、メモなどのメタデータはブラウザ内に保存します。

## GitHub Pagesで使う

GitHub Pagesを有効化すると、スマホから以下のようなURLで使えます。

```text
https://ozasho.github.io/Yuno_movie_album/
```

Google OAuthの承認済みJavaScript生成元には、以下を追加してください。

```text
https://ozasho.github.io
```

## ローカル起動方法

このフォルダでローカルサーバーを起動します。

```powershell
python -m http.server 4173
```

ブラウザで開きます。

```text
http://localhost:4173
```

## 主な機能

- YouTube動画の手動同期
- YouTube URL / 動画IDでの手動追加
- 年月日別の動画一覧
- サムネイル一覧
- 動画詳細での埋め込み再生
- 表示タイトル、撮影日、メモ、タグ、お気に入りの編集
- タイトル、メモ、タグ、年月日検索
- JSONエクスポート / インポート

## 注意点

- ブラウザデータを削除すると、タグやメモなどのメタデータも消えます。
- PCとスマホのメタデータは自動同期されません。
- PCのデータをスマホへ移したい場合は、PCでJSONエクスポートし、スマホでJSONインポートします。
- YouTubeから撮影日が取得できない動画は、投稿日を初期値として使います。
- 非公開動画は視聴や共有の扱いが複雑になるため、限定公開運用を推奨します。
